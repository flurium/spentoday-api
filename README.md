# spentoday-backend

Spentoday backend with ASP.NET Core

To start development clone repository:

```bash
git clone https://github.com/flurium/spentoday-api.git
```

Get inside of folder:

```bash
git checkout dev
```

Add `.env` file to Backend folder with variables:

```bash
DB_CONNECTION_STRING=...

SENDGRID_API_KEY=...
RESEND_API_KEY=...
BREVO_API_KEY=...

JWT_ISSUER=...
JWT_AUDIENCE=...
JWT_SECRET=...

STORJ_ACCESS_KEY=...
STORJ_SECRET_KEY=...
STORJ_ENDPOINT=...
STORJ_PUBLIC_KEY=...

VERCEL_TOKEN=...
VERCEL_PROJECT_ID=...
VERCEL_TEAM_ID=...
```
